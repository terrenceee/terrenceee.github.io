Mac 常用软件记录

作者: Terrenceee's Blog 日期: 2020-05-08 阅读时间:2.3 分钟 字数:617 分类: Mac

Chrome : 丰富的插件系统，快速的反应速度，全球最高占有率的浏览器，吃内存大户。

brew cask install chrome

Surge : Mac 系统中最棒的扶墙软件以及网络调试工具，虽然不是专业人士可能不用不到全部功能，但是依然推荐购买。

brew cask install surge

iiNA : 功能非常强大的播放器，支持很多格式，丝滑流畅，更重要的是它还是个免费软件。

brew cask install iina

ClashX ：同样很棒的扶墙软件，虽然没有 Surge 那么强，但是人家是个免费开源的软件。

brew cask install clashx

Pocket Casts : 老牌播客软件，在2019年转为订阅了，万幸的是在它转为订阅前我曾经购买过网页版本，于是我成为了 LifeTime Member。终身 Plus 会员的好处就是可以可以上传文件来听，这点比 OverCast 和 Castro 要强，并且它对苹果全平台支持都很好。

brew cask install Pocket-Casts

Apple Music : 没啥可说的，最大的流媒体音乐提供商之一，之前订阅过美国的 Apple Music，但是后来有点承受不起每月18元的价格，于是转向了国区。

Spotify :世界用户量第一名的流媒体音乐提供商，跨平台做的很棒，音源丰富，不是会员也可以听，和 Apple Music 补充使用。

brew cask install spotify

LaunchBar :历史悠久的 macOS 启动器，与 Alfred 并称双雄，有很多拓展功能，不过我也只是用做启动器来使用，偶尔会用下 Instant Send 功能，因为学不会 Apple Script。

brew cask install launchbar

Gridea:一款以 Github Page 为基础的静态博客管理应用，是那些想要快速拥有自己私人播客又没有很强动手能力的朋友们的福音，虽然不是一键建立网站的那种类型，但是在整个过程中所有步骤几乎都是可视化的，是我这种小白用户的福音。

brew cask install gridea

balenaEtcher:一款烧录系统到 U 盘和 SD 卡的应用，如果你家里有台树莓派经常需要烧录系统的话可以选择使用这款软件。免费且开源。

brew cask install balenaEtcher

Typora:一款所见即所得的 markdown App，十分美观功能完善而且是免费的，macOS、Windows、Linux 三个平台全方位支持，非常棒的一款 app ，建议下载使用。

brew cask install typora
